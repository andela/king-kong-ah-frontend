Divider Default:

```jsx
<Divider />
```

Divider with all attributes: 
```jsx
<Divider
  width="xxl"
  heigth="zero"
  color="black" />
```

Divider with max-width:
```jsx
<Divider
  width="md"
  />
```

Divider with heigth:
```jsx
<Divider
  height="zero"
  />
```

Divider with color:
```jsx
<Divider
  color="black"
  />
```
