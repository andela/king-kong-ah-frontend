Title Default:

```jsx
<Title>Default Title</Title>
```

Title with all the attributes
```jsx
<Title
  fontSize="normal"
  textAlign="center"
  color="primary"
  display="inline"
  paddingBottom="md"
>
  Example Title
</Title>
```
fontSize Title:

```jsx
<Title
fontSize="medium"
>medium fontSize Title</Title>
```

Grey Coloured Title:

```jsx
<Title
color="grey"
>Grey Coloured Title</Title>
```

Title with small padding bottom:

```jsx
<Title
paddingBottom="sm"
>Padded Title</Title>
```

Primary Colored Title:
```jsx
<Title
color="primary"
>Primary Colored Title</Title>
```

Block Title aligned to center:

```jsx
<Title
display="block"
textAlign="center"
>Block Title</Title>
```
