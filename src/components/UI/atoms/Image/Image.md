Image Default:

```jsx
<Image/>
```

Image with sepcified image url

```jsx
<Image imageUrl='https://assets-ouch.icons8.com/preview/501/57c82ec0-5e6d-423e-b594-5d93a5f5f44b.png'/>
```


Image with sepcified width

```jsx
<Image  width='50%'/>
```

Image with sepcified height

```jsx
<Image  height='200px'/>
```
