Default Button:

```jsx
<Button onClick={() => alert('hello')}>Default</Button>
```

Submit Button:

```jsx
<Button
  buttonType="submit"
  onClick={() => alert('hello')}
>
  Submit
</Button>
```

Get Started Button:

```jsx
<Button
  buttonType="getStarted"
  onClick={() => alert('hello')}
>
  Get Started
</Button>
```

Publish Button:

```jsx
<Button
  onClick={() => alert('hello')}
>
  Publish
</Button>
```

DropDown Button:

```jsx
<Button
  onClick={() => alert('I show stuff')}
buttonType="dropDown"
>
  Publish
</Button>
```
