Text Default:

```jsx
<Text>Default Text</Text>
```

Text with all the attributes
```jsx
<Text
  textTransform="uppercase"
  content='true'
  padding="md"
  fontWeight="normal"
  display="inline"
  fontSize="normal"
  textAlign="center"
  color="primary"
>
  Example Text
</Text>
```
textTransform Text:

```jsx
<Text
textTransform="uppercase"
>textTransform Text</Text>
```
